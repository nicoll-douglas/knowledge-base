# Methods

## Mutable Sequences

| Name        | Description                                                                            |
| ----------- | -------------------------------------------------------------------------------------- |
| `append()`  | Add an item to the end of the sequence                                                 |
| `clear()`   | Remove all items in the sequence                                                       |
| `extend()`  | Add all items in the given iterable to the sequence                                    |
| `insert()`  | Insert the given value at the given index in the sequence                              |
| `pop()`     | Remove and return the item at the given index                                          |
| `remove()`  | Remove the first item that matches the given value or raise `ValueError` if none found |
| `reverse()` | Reverse the sequence in place                                                          |
